# 📸 Sophie Lumière | Portfólio de Fotografia

Este é um projeto de portfólio responsivo desenvolvido para exibir ensaios fotográficos, focado em capturar a essência de momentos e transformá-los em memórias visuais. O site conta com sessões institucionais, galeria categorizada e integração para contato direto.

---

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias web fundamentais, sem a necessidade de frameworks externos, priorizando uma estrutura leve e estilização pura:

* **HTML5:** Estruturação semântica de todas as páginas (`index.html`, `e.html`, `p.html`, `parques.html`).
* **CSS3:** Estilização personalizada, gerenciamento de layouts e design responsivo (`style1.css`).

---

## 📂 Estrutura do Projeto

A organização dos arquivos no repositório segue a estrutura abaixo:

* `/Imagens`: Diretório contendo todas as fotografias e mídias utilizadas no site (Estradas, Parques, Praias e fotos de perfil).
* `index.html`: Página principal contendo a apresentação ("Quem sou eu") e a galeria principal de categorias.
* `e.html`: Página dedicada ao ensaio/categoria "Estradas".
* `p.html`: Página dedicada ao ensaio/categoria "Praias".
* `parques.html`: Página dedicada ao ensaio/categoria "Parques".
* `style1.css`: Arquivo de estilos responsável pelo design visual do portfólio.
